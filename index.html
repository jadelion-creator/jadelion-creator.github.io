Timeline Mixer Documentation

Timeline Mixer extends Unity’s Timeline system with flexible playback control, smooth cross-fading, and centralized timeline management.

Unity’s Playable Director works well for isolated timelines, but it does not provide a built-in workflow for coordinating multiple timelines simultaniously on a single component or implementing seamless transitions between them. Timeline Mixer introduces a system for dynamically blending and controlling timelines from a single component.

Key Features

  • Dynamic cross-fading between timeline assets
  • Blend timelines with animator controllers
  • Centralized management of multiple Timeline assets
  • Cleaner scene hierarchy (no scattered Playable Directors)
  • Minimal setup with a single controlling component
